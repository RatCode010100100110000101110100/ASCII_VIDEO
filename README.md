Hello ! 
This is a simple Ascii video maker with a very ugly front end.
.
Put the 2 code in the same folder and run ascii_gui in terminal like this :
python3 ascii_gui.py
.
Click Browse to select your input video file
Set output filename (like output.mp4)
Change parameters with dropdowns or text entries
Click Convert to start the ASCII conversion
Wait for the message box confirming it finished!
.
Dont forget to install pip library :
pip3 install opencv-python-headless pillow numpy tk
